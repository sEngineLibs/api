class Dispatcher

package se.events

@:directlyUsed@:allow(se.Application)

The Dispatcher class manages event listeners and their execution.

It allows adding and removing event listeners, and updates them every frame.

See also:

Adds an event listener to the dispatcher.

Parameters:

event

A function that returns true when the event condition is met.

callback

The function to be executed when the event triggers.

breakable

If true, the event listener will be removed after execution.

Returns:

The created EventListener instance.

staticremoveEventListener(listener:EventListener):Bool

Removes an event listener from the dispatcher.

Parameters:

listener

The EventListener instance to remove.

Returns:

true if the listener was successfully removed, false otherwise.
